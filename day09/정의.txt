public void call (int x) {

}
------------------------

public - 접근 제어자
	package를 벗어나서도 사용 가능

void - 없음
	return(반환)값이 없음
	--> 반환값이 있다면
		반환되는 데이터 타입 키워드
		int, String, double, int[ ] ……

call - 메소드의 이름(소문자)

( ) - 입력값
	없을 수도 있고, 여러 개일 수도 있음
	
	호출되는 시점에서는 입력값이 중요
	입력값만 다르게 하면, 메소드 이름을 똑같이 사용 가능
	하나의 메소드로 여러개의 기능 구현 가능(다형성, 오버로딩)
